Plugins

Plugins

A plugin provides a set of functionalities you can add to your application. It will bind new components to the Guice context, and may also add scheduled tasks and filters to be able to perform its tasks properly. Each plugin is different and has different needs... Make sure you read the documentation associated with a plugin when you install one!

Note that the plugins with a blue icon are already installed if you use the spincast-default artifact or if you start your application from the Quick Start template! You need to manually install the other ones.

Spincast Routing

The default Router, and other routing related components.

Spincast Request

Components to access HTTP requests information.

Spincast Response

Components to help build an HTTP response.

Spincast Undertow

Undertow is the default HTTP server provided by Spincast. It is modern, performant, and easily embeddable.

Spincast Locale Resolver

Provides an implementation of the locale resolver which is used to find the best locale/language to use.

Spincast TimeZone Resolver

Helps finding the best timezone to use for the current user.

Spincast Variables Add-on

Provides a Request Context add-on to read and write request scoped variables.

Spincast Templating Add-on

Provides a Request Context add-on for easy access to the functionnalities of the Templating Engine.

Spincast Pebble

Provides an implementation for the Templating Engine component, using Pebble.

Spincast Jackson Json

Components to work with Json, using Jackson.

Spincast Jackson XML

Components to work with XML, using Jackson.

Spincast Config

Spincast Dictionary

Allows internationalization ("i18n") in a multilanguage application.

Spincast Http Client
