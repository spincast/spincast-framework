Plugins

Plugins

Most plugins provide a Guice module that you have to install in your application. By doing so, extra components are bound and can be used in your application. Note that the plugins with a dotted line below their title are already installed if you use the spincast-default artifact and the standard Bootstrapper (or if you start from the Quick Start template application)!

Spincast Routing

The default Router, and other routing related components.

Spincast Request

Components to access HTTP requests information.

Spincast Response

Components to help build an HTTP response.

Spincast Undertow

Undertow is the default HTTP server provided by Spincast. It is modern, performant, and easily embeddable.

Spincast Locale Resolver

Provides an implementation of the locale resolver which is used to find the best locale/language to use.

Spincast Variables Add-on

Provides a Request Context add-on to read and write request scoped variables.

Spincast Templating Add-on

Provides a Request Context add-on for easy access to the functionnalities of the Templating Engine.

Spincast Pebble

Provides an implementation for the Templating Engine component, using Pebble.

Spincast Jackson Json

Components to work with Json, using Jackson.

Spincast Jackson XML

Components to work with XML, using Jackson.

Spincast Config

Configuration related components.

Spincast Dictionary

Provides default labels required by any Spincast application.

Spincast HTTP Caching

HTTP Caching management : Etags, Last modification dates, Cache-Control headers, etc.

Spincast Http Client

HTTP Client to easily build and send HTTP requests.

Spincast Http Client
(with Websocket support)

HTTP Client to easily build and send HTTP/WebSocket requests.
